**viteのインストール**
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install 
npm install axios react-router-dom

**css-in-js(Emotion)**
本体
npm install @emotion/react @emotion/styled

開発用ツール（Reactとの連携に必要）
npm install -D @emotion/babel-plugin

**vite.config.tsとtsconfig.app.jsonの書き換え**
```
plugins: [ 
	react(
		{ // これを追加することで、各ファイルで import 忘れても css プロップが使えます
			jsxImportSource: '@emotion/react', 
		}), 
	],


{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "@emotion/react
  }
}
```

**Axios**
npm install axios

**react-router-dom**
npm install react-router-dom
