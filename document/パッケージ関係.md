**初期化**
go mod init hoge

**Ginの導入
go get github.com/gin-gonic/gin

**PostgreSQL用のドライバー**の導入
go get github.com/lib/pq

**依存関係の整理**
go mod tidy
パッケージを追加したら都度実行

**Airの導入**
Dockerfileに下記を記述
RUN go install github.com/air-verse/air@latest

**.air.tomlの作成**
```
root = "."
tmp_dir = "tmp"

[build]
  bin = "./tmp/main"
  cmd = "go build -o ./tmp/main ."
  delay = 1000
  exclude_dir = ["assets", "tmp", "vendor"]
  include_ext = ["go", "tpl", "tmpl", "html"]
  exclude_regex = ["_test.go"]

[log]
  time = true

[color]
  main = "magenta"
  watcher = "cyan"
  build = "yellow"
  runner = "green"
```

**実行コマンドの記述**
CMD ["air", "-c", ".air.toml"]
